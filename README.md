## Prerequisites

- Java 25+
- Node.js 22+

> If you are using nvm: `nvm install 22 && nvm use 22`

> **Java** — install or update via sdkman:
> ```bash
> sudo apt install zip unzip
> curl -s "https://get.sdkman.io" | bash
> sdk install java 25-open
> ```

## Start

```bash
chmod +x start.sh
./start.sh

Application available at: http://localhost:8080

Change port: PORT=9090 ./start.sh

API

Swagger UI: http://localhost:8080/swagger-ui/index.html
```
